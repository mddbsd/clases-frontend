Hoja de Estilo en Cascada (CSS)

CSS, es el acronimo para Cascading Style Sheets que en español significa Hojas de Estilo en Cascada. A diferencia de HTML, que define la esctructura de una pagina web, CSS define la presentación, es decir, cómo deben mostrarse los elementos en la página, su color, su forma, posicion, espaciado, etc. En resumen, la estetica (estilo) del sitio web

No podemos utilizar CSS sin HTML, ya que esta es una herramienta que modifica solo la estética del sitio. Hoy en día, por la manera en que evoluciono la web, tamopoco tiene mucho sentido utilizar HTML sin CSS. Esto lleva a dejar obsoloetas algunas etiquetas.

Muchas etiquetas HTML se utilizan para dar formato a textos, pero estas fueron reemplazadas con la utilización de CSS.

Partes principales de CSS

Hay muchos elementos con los que tenemos que trabajar para declarar los estilos, básicamente los más importantes son:

  • Selectores, mediante los cuales podemos especificar a que elementos de la página estamos haciendo referencia. Los principales selectores son: selector de etiqueta (Solo se escribe el nombre de la etiqueta), selector de clase (caracter .), selector de id (caracter #)
  • Atributos de estilo para definir que cosas queremos estilizar sobre los selectores indicados. Estos se indicant dentro de las llaves {}
  • Una serie de valores, que indican que estilo se debe aplicar a cada atributo sobre cada selector. Algunos valores se pueden expresar en unidades CSS, que sirven para cuantificar estos (pixeles, puntos, pulgadas, etc.). Estos se escriben a la derecha del atributo, luego del símbolo :
