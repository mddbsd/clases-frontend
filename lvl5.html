Diseño de grilla

El módulo de diseño de grilla ofrece un diseño basa en grillas con filas y columnas. Este modulo permite a los desarrolladores crear facilmente diseños web complejos. También facilita la estructura responsiva sin utilizar float o posicionamiento. Las propiedades CSS de grilla son soportadas por todos los navaegadores modernos

Grilla vs. Flexbox

El diseño de grilla debe ser utilizado para diseños bidimensionales, con filas Y columnas.
El diseño Flexbox debe ser utilizado SOLAMENTE para diseños unidimensionales, con filas O columnas

Compontes de grilla CSS

Una grilla siempre consiste de:

  • Contenedor de grilla: el elemento ≶div> padre
  • Item de grilla: son los elementos dentro del contenedor de grilla

Siempre comenzamos con el elemento padre llamado conetendor de grilla, este puede contener uno o varios elementos de grilla. Todos los hijos directos del contenedor de grilla se convierten automaticamente en elementos de grilla

Pistas de grilla (Grid Tracks)

Las filas y columnas de una grilla son definidas con las propiedades grid-template-rows y grid-template-columns (se pueden resumir como grid o grid-template).
Estas definen las pistas de la grilla (grid tracks). Una pista es el espacio entre 2 lineas de grilla adyacentes.

La propiedad grid-template-columns

La propiedad grid-template-columns establece el numero de columnas en el diseño de grilla, y puede definir el anho de cada columna.
El valor de esta propiedad es una lista separada por espacios, donde cada valor individual define el ancho de la respectiva columna.
Si queremos un diseño de grilla con 4 columnas, especificamos el ancho de cada columna como "auto" para que todas las columnas tengan el mismo ancho

Unidad de medida de celdas (fr)

La unidad fr se puede utilizar para definir el ancho de las grillas, de la misma manera que cualquier otra unidad de medida como %, px, em.
La unidad fr significa "fraccion". Esta unidad divide automaticamente el espacio disonible en fracciones.
Ejemplo: 1fr tomara una fraccion del espacio disponible, mientras que 2fr tomara 2 fracciones del espacio disponible

Aqui cada columna tomara un 25% del ancho del contenedor, divido en iguales
